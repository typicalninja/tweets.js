ClientOptions

Options for a client.
Type:
  • Object
Properties:
Name Type Description
subdomain String the subdomain to use for twitter api, ex: api becomes https://api.twitter.com
version String twitter api version to use https://api.twitter.com
consumer_key String your consumer key if authentication as a USER, if not present fetches from env
consumer_secret String Your consumer secret if authentication as a USER, if not present fetches from env
access_token String Your access token (key) if authentication as a USER, if not present fetches from env
access_token_secret String Your access token secret if authentication as a USER, if not present fetches from env
bearer_token String Your bearer token if authenticating as a APP, if not present fetches from env
emitter Boolean weather to enable the stream
emitterUrl String the url to stream from, ex: statuses/filter
autoReconnect Boolean Weather to autoReconnect the stream if a dead stream is found (ping timeout exceeded)
reconnectInterval Number the interval reconnect will take to disconnect and reconnect
verifyCredentials Boolean weather to verify credentials provided on app startup
